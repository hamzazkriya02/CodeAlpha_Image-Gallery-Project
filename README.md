# CodeAlpha Image Gallery Project

A high-performance, responsive image gallery featuring dynamic filtering, real-time search, and a custom lightbox. This project was developed as part of the **CodeAlpha** internship to demonstrate advanced DOM manipulation and modern UI/UX principles.

## 🚀 Features
- **Dynamic Filtering:** Categorize assets (Animals, Tech, Seasons, etc.) instantly.
- **Live Search:** Filter 21+ images in real-time as you type.
- **Interactive Lightbox:** Click any image to view it in full-screen with a "Zoom" effect.
- **Dark/Light Mode:** Full theme support with seamless transitions.
- **Aero Design:** A modern, glassmorphism-inspired interface with shimmer loading effects.

## 🛠️ Tech Stack
- **HTML5** (Semantic structure)
- **CSS3** (Grid, Flexbox, & Animations)
- **Vanilla JavaScript** (Data-driven rendering & Filtering logic)
